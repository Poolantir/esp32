# Source

- **main.cpp** – main app (ToF, LEDs, servo). Built by env `esp32doit-devkit-v1`.
- **bluetooth_test.cpp**, **wifi_test.cpp**, **servo_test.cpp**, **tof_test.cpp** – test programs. Each is built by its own env; see `running_tests.md`.

Keep `test/` and these `*_test.cpp` files in sync when editing tests.
